# [aichat] KoBERT 기반 한국어 뉘앙스 처리 엔진 명세서

## 1. 개요
- 목적: 한국어의 미묘한 표현(뉘앙스)을 분석하여 사용자의 정확한 증상 위급도 및 의도를 파악함
- 역할: RASA NLU의 부족한 한국어 문맥 이해력을 보완하는 감각 기관(입/귀) 계층

## 2. 주요 기능 및 프로세스
- 문맥 분석: 사용자가 입력한 문장이 응급 상황인지 단순 상담인지에 대한 다중 분류(Multi-label Classification) 수행
- 뉘앙스 추출: 콕콕 쑤시다, 띵하다, 뒤틀리다와 같은 한국어 특유의 통증 표현을 고차원 벡터로 수치화
- 의도 전달: 분석된 감정 및 위급도 점수를 RASA Core에 전달하여 대화 흐름의 방향 결정

## 3. 기술적 세부 사항
- 베이스 모델: SKT KoBERT (Bidirectional Encoder Representations from Transformers)
- 전처리 도구: SentencePiece 기반의 KoBERT 전용 토크나이저 활용
- 연동 데이터: 의료 도메인에 특화된 한국어 Q&A 데이터셋으로 파인튜닝(Fine-tuning) 진행

## 4. RASA와의 연동 시나리오
- 단계 1: Flask/FastAPI 브릿지 서버에서 사용자의 원본 메시지 수신
- 단계 2: KoBERT 모델이 메시지를 실시간 스캔하여 위급도(0.0 ~ 1.0) 산출
- 단계 3: 위급도가 0.8 이상일 경우 RASA의 의도 분류와 상관없이 즉시 emergencyAlert 프로토콜 트리거

## 5. 기대 효과
- 오진률 감소: 단순 키워드 매칭의 한계를 벗어나 문장 전체의 의미를 파악함으로써 정확한 정보 제공 가능
- 사용자 친화적 대응: 사용자의 말투에서 느껴지는 불안감을 감지하여 보다 공감하는 형태의 챗봇 페르소나 구현 가능
