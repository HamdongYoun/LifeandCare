# [aichat] 인공지능 의료 대화 시나리오 가이드

## 시나리오 A: 일반 증상 상담 (심각도 낮음)
- User: 머리가 살짝 아픈데 원인이 뭘까?
- AI: 증상에 대한 일반적 원인(피로, 탈수 등) 설명 후 대처법 추천.
- UI: 일반 텍스트 말풍선.

## 시나리오 B: 주변 병원 추천 (심각도 중간)
- User: 무릎을 삐었는데 근처 정형외과 어디 있어?
- AI: 사용자의 좌표를 기반으로 최근 거리순 병원 3곳 노출.
- UI: Hospital Card Slider 위젯.

## 시나리오 C: 응급 상황 고지 (심각도 높음)
- User: 갑자기 가슴이 너무 답답하고 숨쉬기 힘들어.
- AI: 즉시 응급 위반 키워드 감지 -> 심근경색 의심 판단.
- UI: Emergency Alert 팝업 활성화 + 3초 카운트다운 시작.
